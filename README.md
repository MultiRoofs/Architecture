# Architecture
Architecture Repository containing the components for the Multiroofs solution

The repository is contained into an Archimate file. That file can be opened by Archi, a free to use tool.
One can download this tool at: https://www.archimatetool.com/download/

Viewing the contents is easy, just download the archimate file and select 'open' on the Archi menu.
